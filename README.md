**Glow Mart** is an **Android app** for discovering and exploring makeup products. It uses a **free makeup API** to fetch product details and provides a clean, modern interface with **Dark Mode support**.  

Built using **MVVM architecture**, this app demonstrates modern Android development practices with Kotlin.

---

## ðŸŒŸ Features

- **Browse Products:** Explore a wide range of makeup products.  
- **Dark Mode Support:** Switch between light and dark themes.  
- **Firebase Authentication:** Secure login/signup for users.  
- **Efficient Image Loading:** Powered by Glide for smooth UI.  
- **Persistent Settings:** Save user preferences with Shared Preferences.  
- **Asynchronous Data Fetching:** Uses Kotlin Coroutines for smooth experience.  
- **Modern Architecture:** MVVM with ViewModel for clean code structure.
